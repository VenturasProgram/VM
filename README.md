# VM - Aplicativo de Música

Um aplicativo de música feito por um brasileiro 🎵

## 📋 Descrição

VM é um aplicativo moderno e intuitivo para tocar, organizar e gerenciar sua coleção de músicas. Desenvolvido com foco na experiência do usuário, oferece uma interface amigável e recursos poderosos para os amantes de música.

## ✨ Características

- 🎵 Reprodução de música de múltiplas fontes
- 📝 Criar e gerenciar playlists personalizadas
- 🔍 Buscar por músicas, artistas e álbuns
- 🎨 Interface amigável e responsiva
- ⚡ Performance otimizada

## 🚀 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/VenturasProgram/VM.git
```

2. Navegue até o diretório do projeto:
```bash
cd VM
```

3. Instale as dependências:
```bash
npm install
```

## 💻 Como Usar

Para iniciar o aplicativo, execute:
```bash
npm start
```

O aplicativo será aberto em seu navegador padrão.
